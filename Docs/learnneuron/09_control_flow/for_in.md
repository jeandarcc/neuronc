# For-In Loop

Iterate over collections or ranges with `for-in`.

---

## Syntax

```npp
for (element in collection) {
    // body
}
```

---

## Iterating a Collection

```npp
for (item in items) {
    Process(item);
}
```

---

## With Range

```npp
for (i in 0..10) {
    Print(i);
}
```
